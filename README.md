# Car-Rental-System
This is a console-based Car Rental System Using java. <br>
# Features <br>
* Add new Cars <br>
* View available Cars <br>
* Rent a car for a given number of days <br>
* Calculate total rent amount <br>
* Check car availability <br>
* Strore cars data using ArrayList <br>

## Concepts Used -
* Encapsulation <br>
* ArrayList <br>
* Scanner <br>
* Switch-case <br>
* try-catch <br>
* loop (while loop) <br>
* conditional Statements (if-else)
